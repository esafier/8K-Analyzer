Re-Summarize Filings

Re-run the LLM on filings already in the database to fix bad or missing summaries. No need to re-fetch from SEC — uses the stored filing text. Also the only way to populate triage verdicts (Deep Look / Monitor / Pass), signal scores, and forfeiture flags on filings analyzed before those features existed — pick a date range to rate older rows.

This re-sends each filing through the LLM to get a fresh summary and classification. Useful when the LLM was down or credits ran out during a backfill.

Retry Missing Summaries

Re-fetches the SEC document and runs the LLM for filings where the original backfill couldn't download the text (empty summary + hard-coded "Management Change" category). This is the only way to fix rows that "Re-Summarize" can't touch.

Leave dates blank to retry the last 7 days. Watch the Render logs — each row will print "re-fetching SEC text..." then either "Updated" or "Fetch failed".

Backfill Departure History (EDGAR)

For every filing that contains an executive departure, fetches the company's 5.02 filings from SEC EDGAR over the last 24 months and stamps the filing with a departure count and history list. Powers the "N dep/24mo" cluster badge on the dashboard and makes the detail-page departures card appear automatically. New filings get this at ingest — run this once to cover older ones.

Cost: one EDGAR lookup per company plus a small LLM extraction per historical filing (cached — re-runs are nearly free). Safe to re-run; already-stamped filings are skipped.

Retrofit Flags (Market Targets, Departure Counts, Forfeiture)

Scans every existing filing's structured summary and (1) flags those that disclose stock-price, market-cap, or TSR comp targets, (2) stamps the FORFEITS COMP / NO SUCCESSOR flags that power the bearish dashboard filters, and (3) counts each filing's executive departures — which powers the "N departures / 24mo" cluster badge on the dashboard. Zero LLM cost — pure data transform on what's already stored.

Safe to re-run any time — only updates rows where the flag would change.

Refresh Market Caps

Clear Database
